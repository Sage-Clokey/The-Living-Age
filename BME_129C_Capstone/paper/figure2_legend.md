# Figure 2: Biological Protein Interaction Networks Exhibit Decentralized, Robust Architecture

**Figure 2. Topological analysis of biological networks compared to centralized reference architectures.** Three biological networks — *E. coli* protein-protein interaction (PPI, 529 nodes, 6,951 edges; STRING DB), *S. cerevisiae* PPI (573 nodes, 6,342 edges; STRING DB), and *E. coli* gene regulatory network (GRN, 282 nodes, 308 edges; RegulonDB) — were analyzed alongside centralized references: a star graph (single hub connected to all nodes), hub-and-spoke (10 hubs with spokes), and an Erdos-Renyi random graph, all matched to the E. coli PPI network size.

**(A) Degree distribution (log-log).** Biological PPI networks display broad, heavy-tailed degree distributions characteristic of scale-free-like topology, with power-law exponents in the range α = 2.05–2.44. Many nodes have few connections while a small number of hubs have many — but no single node dominates. In contrast, the star graph concentrates all connections in one node (single spike at k = 528), and the random graph shows a narrow bell curve with no hubs.

**(B) Betweenness centrality inequality (Gini coefficient).** All biological networks have high Gini coefficients (0.85–0.94), indicating that information flow is concentrated through hub proteins — consistent with scale-free topology. However, the star graph (Gini = 1.00) routes *all* shortest paths through a single node, while biological networks distribute this load across *multiple* hubs (e.g., ompA in *E. coli*, CDC28 in yeast). The random graph has low Gini (0.20) but lacks the organized structure of biological networks. The key distinction is not whether hubs exist, but whether the system survives their loss.

**(C) Robustness under node removal.** Solid lines show network integrity (giant component fraction) during random node removal; dashed lines show targeted attack (highest-degree nodes removed first). PPI networks maintain >50% connectivity even after removing ~37–49% of nodes under targeted attack and ~49% under random failure. In contrast, the star graph collapses to zero connectivity upon removal of its single hub (1.9% removal threshold). Hub-and-spoke architecture also collapses rapidly under targeted attack (1.9%). This demonstrates that biological networks, despite having hubs, distribute resilience across multiple redundant pathways — the hallmark of a decentralized system.

**Data sources:** STRING database v12.0 (high-confidence interactions, combined score ≥ 700); RegulonDB v11 (curated TF-gene interactions). Reference networks were generated with NetworkX using matched node and edge counts.
